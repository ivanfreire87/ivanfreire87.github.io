  Sobre

O meu nome é Ivan Freire e sou licenciado em engenharia informática e de computadores.

O interesse pela informática despertou muito cedo, quando tive o meu primeiro computador e sempre que surgia algum problema com o mesmo eu tentava sempre resolvê-lo sozinho.

Gostava de alterar o código fonte de páginas web para ver as alterações que ocorriam e fiz o meu primeiro site por volta dos 14 anos.

Sempre fui curioso, com vontade de aprender e adoro desafios, tentando ser o melhor possível no que faço, tanto a nível pessoal como profissional. Estas características pessoais reflectiram-se no meu percurso académico pois, uma vez que este curso não disponibilizava regime pós-laboral, tive de aprender a maior parte das matérias sozinho enquanto conciliava o mesmo com o trabalho.
